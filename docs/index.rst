Asciiville: ASCII Art generation and display
============================================

Welcome to the documentation for `Asciiville`_, a suite of tools to generate
and display ASCII Art along with utilities to enhance a character-based
terminal or console environment.

If you're new to Asciiville, begin with the :doc:`README` README. That
walks you through installing Asciiville, initializing the configuration,
and starting to generate your own ASCII Art or viewing any of the more
than a thousand ASCII Art files included in Asciiville.

If you discover an issue or problem with Asciiville then `file a bug`_ in the
issue tracker. Please let us know where you think this documentation can be
improved.

.. _Asciiville: https://github.com/doctorfree/Asciiville
.. _file a bug: https://github.com/doctorfree/Asciiville/issues
